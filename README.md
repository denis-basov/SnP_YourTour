# SnP_YourTour

## Описание проекта

SnP_YourTour — это веб-сайт туристического сервиса "YourTour", предназначенный для организации идеальных путешествий. Проект включает туры в горы, на пляж, в города, а также возможность создать собственный тур или выбрать готовый. Сайт разработан с использованием современных веб-технологий для обеспечения быстрой и отзывчивой работы.

## Технологии

- **HTML5**: Структура веб-страниц.
- **SASS**: Препроцессор CSS для модульной стилизации.
- **JavaScript**: Для интерактивности (используется библиотека IMask для маскирования ввода).
- **Vite**: Современный сборщик для разработки и продакшена.

## Стилизация

Проект использует SASS для стилизации. Для компиляции SASS в CSS используются следующие команды:

- `npm run sass:build`: Одиночная сборка (разовая компиляция).
- `npm run sass:watch`: Режим наблюдения (автопересборка при изменениях).
