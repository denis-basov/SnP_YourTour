# SnP_YourTour

## Описание проекта

SnP_YourTour — это веб-проект, использующий современные инструменты для разработки. Проект включает в себя HTML, SASS для стилизации и Vite для сборки и разработки.

## Сборщик проекта

Проект использует **Vite** как современный сборщик для быстрой разработки и оптимизации. Vite обеспечивает мгновенную перезагрузку (HMR) и эффективную сборку для продакшена, что идеально подходит для проектов с SASS и будущим добавлением JavaScript.

### Команды сборщика

- `npm run dev`: Запуск dev-сервера с горячей перезагрузкой (доступен по http://localhost:5173/).
- `npm run build`: Сборка проекта для продакшена (выход в папку `dist/`).
- `npm run preview`: Предварительный просмотр собранного проекта.

Vite автоматически обрабатывает SASS и будет готов к интеграции с JavaScript-модулями в будущем.

## Стилизация

Проект использует SASS для стилизации. Для компиляции SASS в CSS используются следующие команды:

### Команды для стилизации

- `npm run sass:build`: Одиночная сборка (разовая компиляция).
- `npm run sass:watch`: Режим наблюдения (автопересборка при изменениях).

После выполнения команд при изменении файлов в папке `SASS/` автоматически обновляется файл `styles/main.css`, который уже подключен в `index.html`.
